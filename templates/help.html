GENERAL

The dashbaord has been built to be connected. This means that interacting with one of the charts would produce effects in the other charts. This is true of all the charts except the Sentiment plot.

The dashbaord was created this way to ease the process of filtering. This unconventional method of filtering allows the user to answer complex questions which would only usually be possible using more complex visualizations.

The main interaction methods here are: clicking and hovering. Clicking on the elements of the visualizations would select it and filter other charts accordingly. Hovering over any chart element, shows detailed information about the numbers behind it.

Video Analytics

Use charts in this section to answer questions based on videos.

Number of Comments per Video

This chart is as simple as possible. It shows the number of commnets for each of the videos watched by the students.

Click on any of the bars to select that particular video and filter other charts to show only information concerning that video.

Multiple selections are possible which filter other charts according to information from all selections.

Any active selection remains in colour while inactive selections are greyed out.

Click the "reset" link to select all videos again i.e. clear all filters.

Number of Comments over Video Timeline

This chart shows student comments at different minutes of the videos. The timeline has been discretized into minutes.

This chart does not support selecting by clicking. It can only be filtered using the "Number of Comments per Video" chart to its left.

Hovering over the points on each line shows number of comments at that point.

Impact of Individual Student Comment

This chart shows the extent to which other students found the comments of a particular student interesting.

The bubbles represent the number of total number of ratings that a student got for his/her comments.

The color of the bubbles represent the number of ratings which were learning triggers. Learning trigger ratings are a subset of other ratings. See ABOUT AVW page.

Clicking on any bubble, selects that student and filters the other charts accordingly.

Hover over a bubble to reveal detailed info about that student.

Student analytics tab

This tab shows charts which focus mostly on the students/ learners. The charts here can be used to filter other charts based on the students.

Number of Comments per Student

This chart simply the total number of comments made by each students.

Clicking on any of the bars selects the students and filters all the other charts with information based on only that student. Useful for gaining insight on just a particular student.

Number of Aspects used by Students

This chart shows the different aspects and how many times they were used throughout by the students.

Click on the bars to select an aspect and filter other charts based on that aspects. This allows the user to gain insight based on that particular aspect.

Student Comment Sentiment per video

This chart is quite different from the rest and not connected to the others. It is an attempt to analyse the actual comments made by the students. It uses a text sentiment analytics service from the Microsoft Azure platform.

Select a student and a video and this this chart shows how positive or negative each of the student's comments are.
